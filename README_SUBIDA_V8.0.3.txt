NATURA VIDA V8.0.3

Registro territorial vinculado y formularios móviles estables.

CAMBIOS PRINCIPALES
- El panel Capas incorpora un botón verde “Continuar” y también se cierra al tocar fuera.
- Los formularios siempre se muestran por encima del mapa y sus controles.
- Al abrir un registro territorial se cierran búsqueda y Capas para evitar bloqueos visuales.
- El campo Tienda/prospecto busca clientes existentes mientras se escribe.
- Al seleccionar un cliente se completan nombre, WhatsApp, ciudad, dirección y coordenadas existentes.
- Si el cliente ya existe, Guardar actualiza su ubicación sin crear un prospecto duplicado.
- Los clientes encontrados sin GPS abren directamente el formulario para completar la georreferencia.

SUBIDA
1. Sustituir todo el contenido del sitio por este paquete.
2. Confirmar que app-version.json indique 8.0.3.
3. Esperar el despliegue de GitHub Pages.
4. Abrir la app y usar Actualizar ahora cuando aparezca el aviso.
5. Probar búsqueda de un cliente existente, captura GPS y guardado.
