PAQUETE LISTO PARA GITHUB PAGES

1. Descomprime el ZIP.
2. Sube TODO el contenido directamente a la raíz del repositorio.
3. No subas la carpeta contenedora como una carpeta adicional.
4. Confirma el reemplazo de los archivos anteriores.
5. GitHub Actions ejecutará:
   - auditoría funcional V8.0.2;
   - auditoría de despliegue;
   - validación de sintaxis de todos los JavaScript;
   - creación del sitio limpio para GitHub Pages.

DESPUÉS DE PUBLICAR
- Abre la aplicación con internet.
- En Administración > Actualización, pulsa Actualizar ahora si aparece la versión anterior.
- Si el navegador conserva una pestaña antigua, ciérrala y vuelve a abrir la aplicación.
- Verifica Inicio, Más, Territorio, una búsqueda de cliente y una vista previa de recibo.

El paquete no incluye SQL. Conserva la base de datos y la configuración Supabase existente.
