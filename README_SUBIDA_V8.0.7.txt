NATURA VIDA V8.0.7 — REGLAS COMERCIALES, MÁRGENES Y DESCUENTOS

1. Descomprime el ZIP.
2. Sube el contenido de la carpeta directamente a la raíz del repositorio GitHub.
3. No subas una carpeta contenedora adicional.
4. Confirma que index.html, manifest.json, service-worker.js y app-version.json queden en la raíz.
5. Espera la ejecución de GitHub Actions.
6. En la aplicación abre Más → Actualizaciones → Actualizar ahora.

PRUEBA RECOMENDADA DESPUÉS DE PUBLICAR

- Comprobar que la cápsula de conexión aparezca pequeña arriba a la derecha.
- Abrir Configuración → Reglas comerciales.
- Revisar los valores y activar manualmente “Sí, proteger precios”.
- Guardar margen mínimo y límites por rol.
- Crear una promoción de prueba con fecha y producto.
- Simular utilidad.
- Intentar una rebaja dentro del límite y otra por debajo del precio mínimo.
- Probar una cuenta de representante y una de vendedor vinculado.
- Confirmar que la venta autorizada se guarde y la venta fuera de regla se bloquee.

IMPORTANTE

La configuración y las ventas utilizan Supabase como fuente oficial. Esta versión no recupera la cola offline automática y no incorpora todavía inteligencia artificial.
