PROTECCIONES
- No se modifica ni elimina una venta original.
- No se fusionan físicamente las deudas.
- No se descuenta inventario por registros históricos.
- No se marca pagada una operación con saldo.
- Los duplicados históricos se omiten mediante historicalImportKey.
- La restauración real de base no se ofrece desde el navegador.
- Toda acción sensible genera auditoría cuando el servicio está disponible.
